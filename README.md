# 오늘의 그림일기

초등학생때 쓰던 그림일기 처럼 그림과 함께 짤막한 생각을 기록할 수 있는 그림일기🎨<br/>

![1](https://user-images.githubusercontent.com/37430920/131306598-f99ae687-bb07-4731-a7be-a318611f9a92.gif)

## 실제 배포 링크
https://drawing-diary.vercel.app/


## 설치 및 실행 방법

```sh
npm install
npm run dev
```
